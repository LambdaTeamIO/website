# Setup
Requirements:

- [Elm](http://elm-lang.org/install)
- [Node](https://nodejs.org/en/download/) version > 6
- [Yarn](https://yarnpkg.com/)

Install node packages:

From project folder `website` run
```
$ cd website
$ yarn
```

# Run it!

In terminal run:

```
$ yarn start
```
